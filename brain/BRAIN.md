# Anodizex Brain

## Purpose
This is the durable project memory for Anodizex. It keeps product direction, architecture context, implementation rules, tasks, and decisions aligned for human and AI collaborators.

## Product Direction
- Anodizex is an aluminium systems company focused on premium windows, sliding systems, doors, façades, and related architectural aluminium products.
- The first implementation phase reuses the copied `after-service` monorepo as a starting point, then removes or adapts follow-up-service concepts during later edit/delete passes.
- Reference companies for positioning and product inspiration: Schüco, Reynaers Aluminium, Kawneer, TECHNAL, and AluK.

## Working Rules
- Read relevant Brain files before meaningful changes.
- Update Brain docs when product scope, architecture, API, database, or task state changes.
- Keep copied code namespaces as-is until a dedicated rename/refactor pass.
