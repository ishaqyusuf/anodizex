# afterservice Overview

## Purpose
This file explains what afterservice is and what system must exist to support it.

## Product
afterservice helps local operators manage after-service follow-up. It turns completed service work into customer check-ins, message drafts, reminders, review requests, issue resolution loops, and repeat-business prompts.

## Audience
- Repair shops
- Installers
- Clinics and wellness operators
- Salons and spas
- Local contractors
- Field service teams
- Any operator whose customer experience continues after the job is done

## Core Workflow
1. Record a customer.
2. Record a completed service job.
3. Create or schedule a follow-up.
4. Work the follow-up board.
5. Log contact, replies, outcomes, and closed loops.

## Success Criteria
- Operators can see all after-service work in one board.
- Every follow-up is tied to a customer and optionally a service job.
- Manual workflows work before automation is introduced.
- Billing controls workspace entitlements cleanly.
