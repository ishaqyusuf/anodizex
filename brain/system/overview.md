# Anodizex Overview

## Purpose
This file explains what Anodizex is and what system must exist to support it.

## Product
Anodizex supports aluminium windows, sliding systems, doors, façades, project showcases, customer enquiries, and project quotations. The copied follow-up foundation remains available while the product is progressively reshaped around Anodizex operations.

## Audience
- Architectural aluminium companies
- Window, door, sliding system, and façade installers
- Project managers preparing customer quotations
- Admin users managing public project content, enquiries, and gallery media

## Core Workflow
1. Publish Anodizex website content, gallery items, project roadmap entries, and blog posts.
2. Capture customer enquiries from the public contact form.
3. Maintain customer and project follow-up records.
4. Manage material costs and build BOQ-style project quotations.
5. Review saved quotes, project media, and customer communication history from the dashboard.

## Success Criteria
- Public website content is editable from the dashboard.
- Contact enquiries are stored and can trigger admin/customer emails when email env is configured.
- Project quotations preserve material cost snapshots and server-calculated totals.
- Workspace-scoped permissions keep Anodizex data isolated per account.
